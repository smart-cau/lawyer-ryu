## 3. 핵심 강점 요약

1. **검사 경력 19년** (공식 표기) — 수사·공판·영장 단계 전 영역 직접 수행, **부부장 검사로 퇴직**
2. **대한변협 등록 형사법 전문 변호사** — 검사 19년 경력에 더해 형사 분야 전문성을 객관 인증받은 자격 (검사 출신 변호사 시장에서 차별화 가능한 권위 신호)
3. **전국 권역 검찰청 근무 경험**
   - 호남(광주지검, 순천지청, 전주지검)
   - 영남(창원지검, 부산지검·서부지청, 대구지검)
4. **성폭력 전담 합의부 공판 다수** (부산 2015~2017, 창원 2019~2021)
   → 공식 포지셔닝 = **"탁월한 성범죄전문변호사"**
5. **검찰 특수·강력·기업재산범죄** 전 영역 직접 수사·공판 경험
   - 기술유출·지적재산권·다중피해범죄·공정거래·전세사기·조세·금융 등
6. **국민참여재판 경험** 다수 (창원·부산)
