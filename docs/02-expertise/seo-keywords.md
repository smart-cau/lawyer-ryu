## 2. 매트릭스 (지역 × 분야) — Top Tier 페이지 후보

> 행=지역, 열=분야. 각 셀은 하나의 **랜딩페이지 또는 블로그 글의 타깃 쿼리**가 된다.
> ⭐ = 1차 우선 페이지(공식 포지셔닝 정합), ◯ = 2차, △ = 3차/블로그.

| 지역 ＼ 분야 | 성범죄 | 특수 | 강력(성범죄 외) | 형사일반 | 기업재산범죄 |
|--------------|:---:|:---:|:---:|:---:|:---:|
| 창원 | ⭐ | ⭐ | ⭐ | ⭐ | ⭐ |
| 마산(합포·회원) | ⭐ | ◯ | ◯ | ⭐ | ◯ |
| 진해 | ⭐ | ◯ | ◯ | ⭐ | △ |
| 김해 | ⭐ | ◯ | ◯ | ⭐ | ◯ |
| 양산 | ◯ | △ | ◯ | ◯ | △ |
| 거제 | ◯ | △ | ◯ | ◯ | △ |
| 통영 | △ | △ | △ | ◯ | △ |
| 부산 | ◯ | ◯ | ◯ | ◯ | ◯ |

## 3. 검색 의도 분류 (Intent Tagging)

> Google는 의도(Intent)에 맞는 페이지를 노출시킴. 키워드를 의도별로 묶어
> 페이지 타입(상담/안내/사례/블로그)을 정확히 매칭한다.

### 3-1. Transactional (즉시 상담·선임 의도) — **CTA 강한 페이지**
- 형태: `[지역] + [분야] + [변호사 / 무료상담 / 추천 / 잘하는]`
- 예시:
  - 창원 성범죄 변호사
  - 창원 성폭행 변호사 추천
  - 창원 성범죄 무료상담
  - 마산 형사 변호사 추천
  - 김해 음주운전 변호사 무료상담
  - 진해 성범죄 변호사
- 페이지 타입: **랜딩(상담 CTA·연락처·약도 강조)**

### 3-2. Commercial-Investigation (비교·검토 의도)
- 형태: `[분야] + [전문 / 잘하는 / 후기 / 경력]`
- 예시:
  - 검사출신 변호사
  - 부부장검사출신 변호사
  - 성범죄전문변호사 후기
  - 검사출신 형사변호사 추천
  - 류남경 변호사 후기
- 페이지 타입: **변호사 소개·경력 페이지 + 후기/사례**

### 3-3. Informational (정보 검색 의도) — **블로그·가이드 콘텐츠**
- 형태: `[혐의명] + [처벌 / 형량 / 절차 / 대응방법 / 공소시효 / 합의금]`
- 예시:
  - 강제추행 처벌 수위
  - 준강간 무죄 사례
  - 성범죄 합의금 시세
  - 카메라등이용촬영 초범 형량
  - 음주운전 2회 처벌
  - 명예훼손 공소시효
  - 압수수색 대응방법
  - 검찰 조사 변호사 동행
- 페이지 타입: **블로그(롱폼 가이드)**, FAQPage 구조화데이터 가능

### 3-4. Navigational (특정 인물·로펌 검색)
- 예시:
  - 류남경 변호사
  - 류남경 검사
  - 법무법인 인유 창원
  - 법무법인 인유 류남경
- 페이지 타입: **About / 인물 페이지 / 로펌 소개**
- ※ "류남경 검사" 검색의 경우 검색결과를 변호사 사이트로 흡수하는 것이 핵심.
  Person/Attorney 구조화 데이터 + sameAs 로 정체성 명확화.

## 4. 차별화 modifier (USP 키워드)

공식 포지셔닝에서 도출된 **본 사이트만 정당하게 사용 가능한** 차별화 키워드.
모든 분야 페이지의 H2·H3·메타에 일관 노출하면 브랜드 키워드 자산이 누적됨.

| modifier | 사용 위치 |
|----------|-----------|
| `검사출신` | 모든 분야 페이지 H1·메타·블로그 |
| `부부장검사출신` | 변호사 소개·경력 페이지 |
| `19년 검사 경력` | Hero·About·전 카테고리 헤더 |
| `성폭력전담합의부 공판 경력` | 성범죄 랜딩 H2 |
| `검사 및 검찰수사관 출신 팀` | 팀 소개 + Hero 보조 |
| `수사하고 증거를 찾아주는 변호사` | Hero 헤드라인(공식 카피) |
| `수사기관을 대신하여 억울함을 풀어드립니다` | 메타·CTA 보조 |
| `직접 상담 및 변론수행` | 상담 페이지 |

⚠️ 키워드 스터핑 금지([title-links 가이드 §Don't](https://developers.google.com/search/docs/appearance/title-link)):
   같은 modifier를 한 페이지에 8회 이상 노출하지 않는다.

## 6. 동의어·롱테일 풀

같은 의미를 가진 한국어 변형을 모두 페이지 본문(H2/H3/문단)에 자연스럽게 노출하면
substring 매칭과 롱테일 커버리지가 동시에 올라감.

### 6-1. 직역·자격 표현
- 변호사 / 형사변호사 / 형사전문변호사
- 검사출신 변호사 / 검사 출신 변호사 / 검사 출신
- 부부장검사 출신 / 전 부부장검사 / 부부장 출신
- 전관 / 전관예우 → ⚠ "전관예우"는 부정적 함의 가능, 직접 사용 지양

### 6-2. 분야 동의어
| 정식 | 일상 표현 |
|------|----------|
| 강제추행 | 추행, 만짐 |
| 준강간 | 술취한 상태 성관계, 의식없는 상태 |
| 카메라등이용촬영 | 불법촬영, 몰카, 도촬 |
| 통신매체이용음란 | 음란메시지, 음란사진 전송 |
| 횡령 | 회사돈 빼돌림, 공금 유용 |
| 배임 | 배임, 회사 손해 끼침 |
| 명예훼손 | 명예훼손, 비방, 허위사실 |
| 사기 | 사기, 속임, 사기범 |
| 음주운전 | 음주운전, 음주, 측정 |

> 페이지 본문에서는 **정식 법률용어 + 일상 표현을 병기**하면 정보형 검색까지 잡힘.

### 6-3. 절차·상황 롱테일
- 검찰조사 동행
- 피의자 신문 변호인
- 영장실질심사 변호사
- 진술거부 어떻게
- 합의서 작성
- 공탁 방법
- 고소취하 효과
- 처벌불원서
