### M-02 USP — 왜 류남경 변호사인가

**2026-05-15 갱신**: USP 항목을 Frame A (행동·역량 통일) 구조로 재구성. 채택본·결정 이력은 [`usp-copy-candidates-2026-05-14.md`](./usp-copy-candidates-2026-05-14.md) 참조.

| # | 핵심 행동 | 의뢰인 입장 질문 |
|---|---|---|
| 1 | 사고 예측 — 수사기관의 다음 단계 예측 | "어떻게 미리 대비하는가" |
| 2 | 직접 수사 — 참고인·계좌·SNS·현장 직접 조사 | "방법이 어떻게 다른가" |
| 3 | 영장 대응 — 영장·체포·압수수색 단계 즉시 대응 | "위급 시점에 어떻게 보호하는가" |
| 4 | **보류** — 변호사 피드백 받아 보완 | (미정) |

- **종전 USP-A·B·C·D·E 5종 폐기**: USP-A(재판 전 종결)는 결과 톤 가드 부담으로 USP에서 제거 → M-06 또는 Cases로 이전 (이전 슬롯 결정 대기). USP-E(구성요건 한 점)는 차별화 약함 + 법리 검증 미통과로 폐기. USP-D(초기 대응)는 USP 1·3에 함의로 흡수됨 — 별도 *시급성 후크*로의 M-06 이전은 유지
- **톤**: "왜 가능한가/원칙" 추상 레이어. M-05의 "어떻게 진행되는가/행동" 구체 레이어와 의식적으로 분리
- **디자인**: 메인 USP 섹션의 그리드·레이아웃은 변경 없음. 내용만 교체

### M-05 Process

- **출처**: `usp.md` §3의 **9단계 프로세스** (수사 6 + 분기 + 재판 3)
- **단계 요약**:
  - **수사 단계**: 1) 대표 변호사 직접 상담 · 2) 의뢰인 1차 조사 · 3) 유리한 사실·증거 발굴 · 4) 경찰·검찰 조사 시뮬레이션 · 5) 경찰·검찰 조사 입회 · 6) 법률의견서 작성·제출
  - **분기 pill**: "수사단계에서 종결되는 경우" (광고 규정 가드레일 — `usp.md` §6 톤)
  - **재판 단계**: 7) 재판 대비 추가 서면 작성 · 8) 재판 대비 시뮬레이션 · 9) 재판 당일 대표 변호사 함께 변론
- **메시지 (단 하나의 takeaway, 2026-05-12 wireframe 1단계 합의)**: "수임하면 — 이렇게 진행됩니다. 변호사가 직접 9단계로 사건을 풀어갑니다."
  - **안 1 (직접 동작) + 안 2 (체계·예측가능성) 하이브리드**
  - "확신"은 카피 표현이 아니라 *구조*(상담→…→변론 9단계 연속)로 만든다. 결과 보장 카피는 회피
- **톤**: 의뢰인 화자 시점 ("내가 사건을 맡기면, 이런 단계로 진행되겠구나"). 단계별 헤드라인은 *행동·체계*, bullet은 각 step의 어필 핵심만 (1~3개 자유, 채우기용 bullet 금지)
- **M-02 USP와의 관계**: M-02는 *역량·원칙 진술* ("왜 가능한가"), M-05는 *행동·체계 진술* ("수임하면 무엇이 어떤 순서로 진행되는가") — 의식적 분리
- **USP-D와의 관계**: M-05 Step 5 (조사 입회)에 USP-D가 *운영 단위*로 흡수됨. M-06은 USP-D의 *시급성·전환 후크* 측면을 별도 layer로 다룸 (중복 아님, 다른 framing)
- **Layout (2026-05-12 wireframe 5단계 합의)**: **sticky-left + scroll-right editorial timeline**
  - 데스크톱(lg+): 12-col → 좌 col-5 (sticky `top-24`) / gap col-1 / 우 col-6 (scroll)
  - 좌측 박스: overline + h2 + takeaway lead (좌측 정렬)
  - 우측 timeline: 수사 phase 마커 → Step 1~6 → 분기 pill → 재판 phase 마커 → Step 7~9. number badge circle + 헤드라인 + bullet 본문
  - 모바일(lg 미만): grid-cols-1, sticky 해제 (좌측 박스 상단, timeline 하단)
  - 작은 step 라벨(사실/증거/제출/법리 등) **없음** — phase 마커 2개로 충분
  - CTA: v1 보류 (M-06가 conversion 담당)
  - 종결 결과 pill (재판 단계 끝의 "선고" 류): v1 보류 (광고 규정 — `usp.md` §6)
