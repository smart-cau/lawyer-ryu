## 1. 1차 페르소나·정서 전제

- **1차 청자**: 이미 사건이 진행 중인 의뢰인 (피고소·피의자 측 70 : 피해자 측 30 — 변호사 본인 진술)
- **핵심 정서**: "억울함" — 공식 헤드라인이 직접 짚고 있는 후크
- **메인 카피의 주어·관점**: 피의자 톤 중심. 피해자 동선은 services IA 안의 *피해자 보호·고소 대리* leaf에서 처리
- **시급성 페르소나**(당장 경찰 조사 앞둔 의뢰인) 비중 미상 → 별도 큰 슬롯 대신 M-06에서 *Contact 브릿지* 형태로 처리

## 3. 슬롯별 디테일

### M-01 Hero

- **공식 헤드라인** (그대로): "19년 검사 경력으로 수사하고 증거를 찾아주는 변호사! / 수사기관을 대신하여 억울함을 풀어드립니다."
- **보조 카피**: `brand-positioning.md` §2의 보조 카피 후보군에서 선정
- **핵심 자격 흡수**: "변협 등록 형사법 전문 / 19년 검사 / 부부장" — 별도 Trust 슬롯을 두지 않고 Hero 안에서 노출
- **상세 경력**: `/about/lawyer` 페이지에서 처리 (메인 미노출)
- **지역성 노출**: 형태·문구 미정 (구체 문장으로 적지 않음 — 카피 단계에서 다시 결정)
- **CTA 버튼**: v1에서 **보류**. SoT는 010-7552-0301 CTA를 권고하지만 메인 Hero에서는 일단 빼고 출시. 시급한 의뢰인은 헤더 메뉴·M-06·M-07로 도달

### M-02 USP — 왜 류남경 변호사인가

`usp.md` §2의 USP 5종 중 4종을 메인 카드로:

| 카드 | 메시지 축 | 의뢰인 입장 질문 |
|---|---|---|
| USP-A | 재판 전 종결 | "결과가 어떻게 다른가" |
| USP-B | 직접 수사 | "방법이 어떻게 다른가" |
| USP-C | 수사 흐름 예측 | "왜 가능한가" |
| USP-E | 구성요건 한 점 | "전략이 어떻게 다른가" |

- **USP-D는 M-06으로 이동** (시급성·전환 후크 성격이라 다른 layer)
- **톤**: "왜 가능한가/원칙" 추상 레이어. M-05의 "어떻게 진행되는가/행동" 구체 레이어와 의식적으로 분리

### M-03 대표 사례

- **데이터 원천**: Cases 컬렉션
- **렌더링 조건**: 시드 0건이면 섹션 자체 hidden. 등록되면 자동 노출
- **v1 출시 시점**: 시드 미확보 → 초기에는 비노출 상태. 콘텐츠 확보 시점에 활성화
- **카드 수**: 추후 결정 (가장 최근 또는 분야 균형)

### M-04 업무분야

- **단위**: services IA 10 leaf (`docs/02-services/01-services-ia-v1.md` §1)
- **균질 노출**: 형사 8 + 비형사 2 모두 동일한 카드 사이즈·톤
- **"조사·구속 대응" leaf 시각 강조 X** (그 메시지는 M-06에서 풀버전으로)
- **Hero에서 분야 검색 진입자가 빨리 만날 수 있도록 Process(M-05) 앞에 위치**

### M-05 Process

- **출처**: `usp.md` §3의 4단계 프로세스
  1. 의뢰인 1차 조사 (수사관 시점 신문·코칭)
  2. 유리한 사실·증거 발굴 (참고인·디지털 증거 직접 조사)
  3. 증거 10건+ 확보·제출
  4. 관련 판례 제시
- **톤**: "수임하면 무엇이 산출되는가" — 행동·결과 진술
- **M-02 USP-B 카드 카피와의 관계**: M-02는 *역량 진술*("검사 19년 노하우로 사실관계를 직접 재구성합니다"), M-05는 *행동 진술*("참고인을 사무소로 직접 불러 신문하고 카카오톡·계좌·메타데이터를 분석합니다") 식으로 의식적 분리

### M-06 시급성 후크 (Contact 브릿지)

- **흡수 USP**: USP-D (초기 대응)
- **카피 축**: "지금 경찰 조사 앞두고 있다면 — 혼자 가지 마세요"
- **콘텐츠**: 헤더 후크 + 1차 대응 가이드 2~3줄 + (CTA 또는 Contact 인접 처리)
- **"조사·구속 대응" leaf 연계**: 추후 결정. 단순 deep link / leaf 카드 임베드 / 별도 가이드 컴포넌트 등 옵션 보류
- **위치 의도**: Process까지 다 읽고 결정을 미루던 의뢰인의 마지막 푸시. funnel 끝의 brige

### M-07 Contact

- **콘텐츠**: 약도 + 영업시간 + 010-7552-0301
- **폼 없음** (전화 채널 단일화)
- 사무소 좌표·영업시간 등 미확보 항목은 `brand-positioning.md` §7 Open Items 참고
