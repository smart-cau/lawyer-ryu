# Services Leaf IA v0.1

## 1. 1차 페르소나·정서 전제

- **1차 청자**: 외부 검색 직접 진입자. 예 — 네이버·구글에서 "강제추행 변호사", "통매음", "딥페이크 변호사" 검색 → 검색 결과에서 leaf 페이지로 *바로* 진입
- **진입 비중 가정**: services hub 경유보다 외부 검색 직접 진입이 *높다고 가정*. 
- **services hub IA v0.1 §1과의 차이**: hub는 "외부 SEO 진입자 비중 낮음" 가정으로 USP 재노출을 뺐으나, leaf는 정반대 가정. leaf는 *이 페이지가 변호사를 처음 만나는 자리*임을 전제
- **핵심 정서**: 검색 결과 클릭으로 들어온 의뢰인이 5단계 흐름 안에 "내 사건이 여기 있고, 이 변호사가 잘하고, 지금 연락하면 된다"고 확신할 수 있어야 함. 겪고 있는 정서: 억울함, 걱정스러움
