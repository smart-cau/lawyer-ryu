## Product Overview

- **One-liner**: 검사 19년 경력의 형사법 전문 변호사가 창원에서 직접 운영하는 단독 변호사 사무소.
- **What it does**: 형사 사건(피의자·피해자)을 중심으로 수사 단계에서 무혐의·기소유예 종결을 지향하는 변론을 수행. 변호사가 직접 사실관계·증거를 능동적으로 수사·정리하고, 첫 경찰 조사 시뮬레이션·입회·법률의견서 제출까지 9단계 프로세스로 진행한다. 형사 외에 민사·가사·행정 일부도 수임(보조 영역).
- **Product category**: 지역 형사전문 변호사 사무소 (창원·경남, "검사 출신 변호사" 카테고리).
- **Product type**: 변호사 서비스(사건 수임 기반, 오프라인 상담 중심). 홈페이지는 lead-gen 채널.
- **Business model**: 사건당 수임료. 가격은 사건별 협의(웹 비공개). 무료 상담 표기 금지(광고규정).
- **운영 형태**: 류남경 대표변호사 *단독 운영*. 어시스턴트·소속 변호사·수사관 출신 팀 부재. 상담부터 변론까지 본인이 직접 수행.

## Target Audience

- **Target persona (1차)**: 형사 사건이 이미 진행 중인 개인 (피의자 측 70 : 피해자 측 30 — 변호사 본인 진술 기준).
- **Target persona (2차)**: 법인 대표·임원 (기업 사기·금융·산업안전·기술유출 등), 정치인·공직자(반부패), 교사·학부모(학교폭력), 자영업자(영업·인허가 행정).
- **Geo**: 창원·경남·부산권 중심. 사무소 위치 = 경남 창원시 성산구.
- **Decision-makers**: B2C는 의뢰인 본인 + 가족(배우자·부모). B2B 기업 사건은 대표·임원 + 회사 법무팀.
- **Primary use case**: "지금 경찰 조사를 앞두고 있는데 어떻게 해야 하는가" — 첫 조사 전 변호사 동행 결정.
- **Jobs to be done**:
  - **JTBD-1**: 사건이 외부에 알려지기 전에 수사 단계에서 끝내고 싶다 (무혐의·기소유예).
  - **JTBD-2**: 첫 경찰 조사에서 잘못된 진술이 조서에 남지 않도록 변호사 입회를 받고 싶다.
  - **JTBD-3**: 변호사가 변론만 하는 게 아니라 *유리한 증거를 직접 찾아주길* 원한다.
  - **JTBD-4**: 기소되더라도 공판 단계에서 검사 출신 시야로 대응받고 싶다.
- **Use cases (분야별)**: 성범죄(7분류) · 기업 관련 범죄(4분류) · 재산범죄(5분류) · 음주운전·교통(3분류) · 아동학대·학교폭력(2분류) · 일반 형사(8분류) · 반부패(5분류) · 조사·구속 대응(3분류) · 민사·가사(4분류) · 행정(3분류). 자세히는 `docs/02-services/01-services-ia-v1.md`.

## Problems & Pain Points

- **Core problem**: 형사 사건을 처음 겪는 의뢰인은 *수사 단계가 결과를 결정한다*는 사실을 모른 채 "재판에서 다투면 된다"고 인식한다. 그 사이 수사기관은 첫 조사로 증거를 굳혀 둔다 — 이 비대칭을 모르고 들어간 의뢰인은 *변론할 거리가 사라진 상태*에서 변호사를 찾는다.
- **Why current solutions fall short**:
  - 일반 변호사는 *변론* 경력은 있어도 *수사*를 직접 해본 경험이 부족 → 수사 단계 대응의 깊이가 다름.
  - 검사 출신 변호사가 많아졌으나, *변협 등록 형사법 전문* 자격까지 갖춘 사례는 제한적.
  - 대형 로펌은 어시스턴트·소속 변호사 위임으로 진행 → 대표 변호사가 첫 상담부터 재판까지 직접 끌고 가는 일관성이 약함.
- **Costs**:
  - 시간·비용: 공판 진입 시 수개월~수년 + 변호사 비용·심리적 압박 누적.
  - 사회적 평판: 직장·가족·지역 사회 영향 — *기소 자체*만으로 흔들림.
  - 전과 위험: 같은 결론이라도 *언제 도달하느냐*가 의뢰인 삶에 미치는 영향이 다름.
- **Emotional tension**: "억울하다" — 공식 헤드라인이 직접 짚는 정서. + 첫 조사 직전의 *공포·정보 부족*.

## Competitive Landscape

- **Direct**: 창원·부산권 *검사 출신 형사전문 변호사*. → 같은 카테고리 안에서 변협 형사법 전문 등록 + 대표 단독 직접 수행 + 9단계 프로세스 명문화로 차별.
- **Direct**: 창원·부산권 *대형/중형 로펌 형사팀*. → 대표 변호사 일관 수행 vs. 어시스턴트 위임 구조의 차이를 강조.
- **Secondary**: 일반 변호사(변론 중심). → "수사를 해본 변호사"와 "변론만 해본 변호사"의 출발선 차이.
- **Secondary**: 법률 상담 플랫폼(로톡 등). → 사건 의뢰의 *진입점*은 될 수 있으나 *대표 변호사 직접 수행*과 다른 layer.
- **Indirect**: "혼자 조사 받기 / 가족 조언 / 인터넷 검색". → 첫 조사에서 굳어진 진술을 뒤집기가 어렵다는 비대칭을 짚어야 함.

> 결과 비교·우위 주장·경쟁자 직접 비교 카피는 광고규정 위반(§0 #9). 본 경쟁 분석은 *내부 포지셔닝용*이며 페이지 카피로 직접 박지 않는다.

## Differentiation

- **Key differentiators** (자료조사 메모 기준, *카피가 아닌 사실 진술* 단위):
  1. **검사 19년 + 부부장 검사 퇴직 (2024-02)**. 영남권(부산·창원·대구) + 호남권(광주·순천·전주) 권역 근무 경험.
  2. **대한변협 등록 형사법 전문 변호사**. 검사 경력 + 변협 공식 인증의 결합 (검사 출신 변호사 시장에서 상대적 차별).
  3. **성폭력 전담 합의부 공판 다수** (부산 2015~2017, 창원 2019~2021) → "탁월한 성범죄전문변호사"(공식 USP) 포지셔닝의 자산 근거.
  4. **대표 변호사 단독 운영** → 첫 상담부터 재판 당일 변론까지 본인 직접 수행, 어시스턴트·소속 변호사 위임 없음.
  5. **9단계 프로세스 명문화** (수사 6 + 분기 + 재판 3). 의뢰인이 *수임하면 무엇이 어떤 순서로 진행되는지*를 예측 가능.
  6. **변호사 직접 수사 6대 활동**: 참고인·증인 조사, 양형자료, 계좌·거래내역, 디지털 증거, 현장 검증, 법리 주장.
- **How we do it differently**: "변론"이 아니라 "수사기관과 별개의 수사"로 사건 기록 자체를 재구성. 첫 경찰 조사 *전*에 신문 시뮬레이션 → 조사 *중* 입회 → 조사 *후* 법률의견서로 구성요건의 가장 약한 한 점을 흔든다.
- **Why that's better**: 변론은 *주어진 기록 위에서 다툰다*. 직접 수사는 *기록 안에 무엇이 들어가는지*를 바꾼다. 출발선 자체가 다르다.

| "창원 사무소인데 다른 지역에서 의뢰 가능한가?" | 가능. 부산·창원·대구 권역 근무 이력으로 영남권 의뢰 가능. 단, *영향력·연고 암시 금지*(광고규정 §1 #4) — *지식·기술* 프레임으로만 |
| "혼자 운영한다는데 대형 사건 처리 가능한가?" | 대표가 직접 끝까지 수행한다는 사실이 강점. 위임 구조의 일관성 손실을 피한다는 점 강조 |

**Anti-persona**:
- 결과 보장을 카피·발화에서 받고 싶어 하는 의뢰인 (광고규정 충돌, 수임 자체 부적합).
- "변호사 빽으로 검찰 선처" 류의 *영향력·연고*를 기대하는 의뢰인 (광고규정 §1 #4 + 윤리 문제).
- 사건 규모 < 변호사 수임료 인 경범죄 단순 의뢰 (시장 적합성 낮음 — 단, *수임 거절*이 아니라 자가 판단 안내 형태로).

## Switching Dynamics (JTBD Four Forces)

- **Push (현 상태의 불만)**: 첫 조사를 앞두고 정보 부족·공포. 일반 변호사 상담에서 "재판에서 다투면 된다"는 답을 듣고 *수사 단계 대응 깊이*에 의구심. 대형 로펌 상담 후 *어시스턴트 위임* 구조에 불안.
- **Pull (당사로 끌리는 이유)**: "수사단계에서 끝낸다"는 명확한 방향성. 변호사 직접 수사 + 9단계 프로세스의 예측 가능성. 검사 19년 + 변협 형사법 전문 자격의 자산 결합.
- **Habit (현 상태에 머무는 관성)**: "재판에서 다투면 된다"는 일반 인식. 변호사 선임 비용에 대한 망설임. *수사 단계에서 끝난다는 발상* 자체가 일반 의뢰인에겐 새로움.
- **Anxiety (전환에 대한 두려움)**: "결과 보장이 없는데 비용을 미리 지불해야 하나". "혼자 운영하는 사무소가 대형 로펌보다 약하지 않을까". 첫 상담을 위해 사무소에 가는 행위 자체의 심리적 장벽 → 전화 단일 채널(055-719-0302)의 가벼움이 완충.

  - "수사기관을 대신하여 억울함을 풀어드립니다"
  - "다년간의 수사와 공판수행으로 판사·검사의 심리를 꿰뚫는 변호사"
- **변호사 본인 발언 (USP 자료조사 원문, *카피 후보가 아닌 기록*)**:
  - "재판까지 안 가게 만든다 — 수사단계에서 끝내는 형사 변호사"
  - "수사단계에서 끝내고 싶다면 내게 오라"
  - "형사 사건은 수사를 해본 경력 있는 변호사에게 의뢰하라"

- **Words to use**: 수사 단계, 무혐의, 기소유예, 불송치, 불기소, 구성요건, 법률의견서, 첫 조사, 시뮬레이션, 입회, 직접 수사, 변협 등록 형사법 전문, 부부장 검사
- **Words to avoid**:
  - 결과 보장: "반드시", "100%", "확실히", "보장", "성공" → *경로·지향*으로 대체.
  - 최상급: "최고", "1위", "전국 최강" → 광고규정 §0 #2.
  - 수치 비교: "○○% 무혐의", "○○건 승소" → 광고규정 §0 #3.
  - 경쟁자 비교: "다른 변호사와 달리" → 광고규정 §0 #9.
  - 연고·영향력: "검찰 인맥", "검사 동기" → 광고규정 §1 #4.
  - 무료 상담: "무료 상담" → 광고규정 §1 #13.

- **Glossary**:

| 용어 | 의미 |
|------|------|
| 불송치 | 경찰 단계에서 검찰로 사건을 넘기지 않고 종결 (혐의없음 등) |
| 불기소 | 검찰이 기소하지 않고 종결 — 혐의없음 / 죄가안됨 / 공소권없음 |
| 기소유예 | 혐의는 인정되나 정상 참작해 검찰이 기소하지 않는 처분 |
| 구성요건 한 점 | 범죄 성립의 법정 요건 중 가장 약한 한 요소에 자원 집중 |
| 법률의견서 | 변호사가 수사기관에 제출하는 사실·증거·법리 통합 의견서 |
| 9단계 프로세스 | 수사 6단계 + 분기 + 재판 3단계의 류 변호사 수임 표준 흐름 |
| 변협 등록 형사법 전문 변호사 | 대한변호사협회 인증 형사법 분야 전문 자격 |
| 부부장 검사 | 부장검사 직전 직급. 19년 경력 검사의 일반 퇴직 직급 중 하나 |

## Brand Voice

- **Tone**: 신중·정중·전문. 시급성 톤은 제한적 슬롯(시급성 후크·조사·구속 대응 leaf)에서만. 일반 슬롯은 *침착하고 단정한* 변호사 화법.
- **Style**: 의뢰인 화자 시점("내가 사건을 맡기면 이렇게 진행된다"). 사실·구조·자산 근거 진술 우선. 감정·약속·결과 보장 진술 회피.
- **Personality (3~5 형용사)**: 신중한 · 직접적인 · 체계적인 · 권위 있는 · 인간적인.
- **광고규정 톤 가드**:
  - *결과* 진술은 *경로·지향*으로 대체.
  - *우월성* 진술은 *사실 진술*로 대체 (자격·경력·운영 형태).
  - *경쟁자 비교*는 *내부 차별화*로 분리 (페이지에 박지 않음).

## Proof Points

- **Metrics (광고 가능 표현 한정, 출처 명시 가능)**:
  - 검사 재직 19년 (2006~2024.02, 부부장 검사로 퇴직).
  - 사법시험 44회 / 사법연수원 35기.
  - 대한변협 등록 형사법 전문 변호사.
  - 성폭력 전담 합의부 공판: 부산 2015~2017, 창원 2019~2021.
  - 9단계 프로세스 (수사 6 + 분기 + 재판 3).
- **Metrics (사용 금지)**: "○○% 무혐의", "○○건 승소", "최단 종결" 등 수치 비교·우위 표현 (광고규정 §0 #3).
- **Notable cases**: Cases 컬렉션 시드 미확보 상태 → 사이트 출시 시점에 *비노출*로 시작, 콘텐츠 확보 시 활성화 (main IA M-03 정책).
- **Testimonials**: 현재 확보분 없음. 광고규정상 *체험기·후기 광고* 제한 항목 확인 필요.
- **Press**: 법률신문 2024-04-18 영입 보도 외 검찰 인사 기사 5건 (자세히는 `docs/04-cases/media/articles.md`).
